# Adaptive Academic Planner 🎓

An intelligent browser-based academic planning system built using the ReAct Agentic AI framework.  
The project demonstrates how adaptive scheduling and planning agents can be implemented entirely using Vanilla JavaScript without requiring any backend or external APIs.

The system dynamically generates prioritised study schedules, tracks syllabus completion, adapts to disruptions such as skipped sessions or new assignments, and visualises all activities through an interactive calendar interface.

---

## ✨ Features

- ReAct-style Agent Simulation
- Priority-based Study Scheduling
- Deadline Tracking System
- Adaptive Re-planning
- Catch-up Plan Generation
- Interactive Calendar
- Progress Tracking Dashboard
- Session Logging
- Constraint-aware Scheduling
- Dark Themed Responsive UI

---

## 🧠 ReAct Workflow

```text
Thought → Action → Observation → Final Answer
```

The planner simulates an agentic workflow where the system:
1. Analyses deadlines and constraints
2. Invokes specialised planning tools
3. Observes tool outputs
4. Generates an adaptive study plan

---

## 🛠️ Tools Implemented

| Tool | Function |
|------|----------|
| SubjectSchedulerTool | Generates weighted study schedule |
| DeadlineTrackerTool | Detects urgent subjects |
| StudyLoadCalculatorTool | Calculates study distribution |
| ConstraintAdjusterTool | Handles blocked days and holidays |
| AdaptiveReschedulerTool | Replans after disruptions |
| CatchUpPlannerTool | Creates recovery schedules |
| ProgressTrackerTool | Tracks completion statistics |
